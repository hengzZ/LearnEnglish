动名词和不定式 （“谈论观点”）
```
动名词和不定式用于谈论复杂的观点。
▪ 动名词 （动词 + -ing）
Oscar enjoys cooking dinner for Frank.
▪ 不定式 （to + 动词）
Frank plans to eat out.
常用的表达：
    动词 + 动名词/不定式
```
表达 “能力” 的方式
```
▪ 使用 can 搭配动词描述能力 （can 的两种否定形式： can't 和 cannot）
John can drive.
▪ know how to 表示具有做某事的知识或能力 （添加 not 表示否定意义）
I know how to drive.
▪ good at 和 bad at 描述能力程度 （常与 really、 very、 pretty 等副词搭配）
Carol is really good at chess.
▪ terrible at 描述非常差的能力
I'm terrible at driving.
```
询问和描述 “可选项”
```
▪ 询问他人的计划/想法
what do you want to do?
what are we going to do tonight?
what's going on?
▪ 列举可选项 （there is 和 there are 搭配 and）
There's a new movie at the theater and a lecture at the university.
▪ 使用 can 和 could 介绍可能的活动，搭配 or 提出不同的选择
You could go swimming or you could go hiking.
▪ 使用 what else 询问更多选择
what else is there to do? / what else is going on?
```
“时间和地点”
```
三个时间和地点介词 in、 on、 at.
▪ 大的地区、长的时间，比如：国家和年，使用 in。
Beijing is in China.
▪ 街道和较小的地区、星期和日期，使用 on。
I live on Main Street.
▪ 地址、公司、建筑、确切时间，使用 at。
His house is at 465 Pine Street.
▪ （固定搭配） in 搭配 morning、afternoon、evening， at 搭配 night、midnight、noon。
```
“请求” 帮助
```
▪ Could you 问句请求 (同意回答： Sure，No problem，OK. 否定回答： I'm sorry, but...I can't help you.)
Could you do me a favor?
```
阅读技巧（optional）
```
阅读一般信息时，可以尝试快速浏览信息，寻找关键词。（不要停留在陌生词汇）
```
询问 “原因”
```
▪ 使用疑问词 why 询问原因，使用 because 说明原因。
Why did you go to Paris?
Because it's beautiful.
```
建议 （“提出活动建议得几种方法”）
```
▪ Let's + 动词原形
▪ How about 或 What about + 动名词
▪ We could + 动词原形
▪ Why don't we + 动词原形
```
一般现在时表示的将来时 (时态)
```
▪ 谈及的演出活动隶属于某计划表、时间表、演出日程，这种情况最为常见。
The dance performance begins at 8 o'clock tonight.
The movie is over at 9:15 p.m.
```
表达 “愤怒” 的方式
```
▪ 重复
OK, OK. I'm coming. I'm coming.
▪ 表示怀疑
I can't believe it!
▪ 夸张 （使用 always 和 never 来夸大，并重读）
Jake always forgets.
▪ 强烈感情的形容词 （stupid、crazy）
This is stupid!
Are you crazy?
```

形容词的顺序 （“组合多个形容词”）
```
▪ 大小、年龄、颜色 （依次）
He's a thin, middle-aged man.
```
“谈论” 他人
```
▪ 询问概况 （What is (person) like? 或 What does (person) look like?）
▪ 询问特定信息 （What's his hair like? 或 What color are his eyes?）
▪ Good-looking / Attractive 可用于描述男性和女性。
▪ 女性专有描述 Beautiful。
▪ 男性专有描述 Handsome。
```
描述 “关系”
```
▪ 使用动名词 （verb + -ing） 描述一段关系中正在发生的事情。
I always have fun talking with him.
▪ 询问两人间的关系 （How are you and your brother doing? What do you and Sylvia do for fun?）
▪ you two 表达密切的关系
Are you two having trouble?
```
‘too’ 和 ‘enough’
```
▪ too + 形容词， 表示超过/过分。
▪ enough + 形容词， 表示满意/满足。
▪ not + 形容词 + enough， 表示不尽人意。
```
“提建议” (发表个人意见)
```
▪ I think that 表达个人意见。
▪ so do I. / neither do I. 表示赞同。
▪ 征求意见。 （what do you think about...? 或 Do you think...?）
```
‘so’ 和 ‘such’
```
▪ so + 形容词， 或者 so + 副词。 （表示强调）
▪ such + 名词。 （表示强调）
```
